# Icons

Place icon files here:
- `icon16.png` (16x16)
- `icon48.png` (48x48)
- `icon128.png` (128x128)

You can generate these from any icon generator or use the SVG below as a source:

```svg
<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 128 128">
  <rect width="128" height="128" rx="24" fill="#2563EB"/>
  <text x="64" y="82" font-family="system-ui" font-size="64" font-weight="bold" fill="white" text-anchor="middle">PP</text>
</svg>
```
